# Youtube Logo Replacer
Заменяет логотип YouTube на ваш текст

# Политика конфиденциальности

Это расширение не собирает, не хранит и не передаёт никакие данные.  
Все настройки сохраняются **локально** в вашем браузере через `browser.storage.local`.  
Никакие данные не отправляются на серверы, включая Google, Mozilla или третьи стороны.
